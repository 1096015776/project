### 使用
- CDN 直接调用
  ```html
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/cn-fontsource-fz-kai-z-03-regular/font.css"></link>
  <style>
    html{
      font-family: Roboto, "FZKai-Z03", sans-serif;
    }
  </style>
  ```
- 本地化
  > npm i cn-fontsource-fz-kai-z-03-regular

  > 加载: font.css
